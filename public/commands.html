Command list

Below is the full list of Polaris' commands. Click a title to show more in-depth information. Arguments with [ ] are required while those with { } are optional. Do not include these brackets when running commands.

Please note that the prefix may be different in each server. You can mention the bot instead.

Description

The help command lists all of Polaris' commands.If a command is provided, it will give you in-depth information about that command.

Usage

.help {Command name}

Description

Allows you to link your ROBLOX account to your discord one. You will need to place a code in your ROBLOX Profile. After putting the code there, you need to run done.

Usage

.linkaccount [ROBLOX name]

Aliases

verify, link

Description

Completes a link account process by checking your ROBLOX Profile for the code. You must run the linkaccount command before using this.

Usage

.done

Description

Fetches your roles according to the server settings. This will check your rank in the set groups and give you roles if you should be given them. You must link your account prior to use. Also updates nicknames.

Usage

.getroles

Aliases

getrole

Description

Allows you to edit server settings. Only one prompt can be run at a time. You must have admin in your discord server to run this command.

Additional info

When a menu appears, you should say the keyword on the left which is in bold, for example, server. This will allow you to navigate the menu.

Usage

.settings {Settings group}

Description

This command will create discord roles based on the ranks in a ROBLOX Group. You can provide a Group ID, or it will use the server settings' id by default. If neither are set, it will ask for one. This is ideal for setting up new discord servers.

Usage

.setup {Group ID}

Description

Updates a user's roles based on their group rank, like getroles, only it can be run by anyone with manage roles permission, on other users. Also updates nicknames.

Usage
